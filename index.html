jQuery?

  • Lightweight (24kb minified and gzipped), Cross-browser JavaScript library
  • Aimed to increase productivity by offering an extended set of shorthand functions.
  • Created by John Resig in 2006, currently working @ Mozilla Corp
  • Open Source (MIT or GPL license)

Some numbers...

  • Current version: 1.4.2 (3x speedup since last year version)
  • Used by 35% of the 10.000 top sites
  • 69.4% "market share"

Why using jQuery

  • We have the standards issued by the W3C
  • JavaScript, DOM, CSS implementations of the standards can differ (right IE?)
  • To avoid writing specific hacks for some browsers : need an abstraction library
