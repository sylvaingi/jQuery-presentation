jQuery?

  • Lightweight (24kb minified and gzipped), Cross-browser JavaScript library
  • Aimed to increase productivity by offering an extended set of shorthand functions.
  • Open Source (MIT or GPL license)
  • Created by John Resig in 2006, currently working @ Mozilla Corp
  • Current version : 1.4.2

Why jQuery?

  • 100% compatible with IE 6+,Firefox 2+, Safari 3+,Opera 9+ and Chrome
  • Simple API
  • Great documentation
  • Lots of plugins
  • Big community

Basics, find something : Selectors

The $ function, your new friend :

$("body") get <body> element

$("#someid") get element with id someid

$("div.myclass") get all <div> with class my class

$("input[name=toto]") get all input with toto name

$("tr:first") get 1st row

$("#scroll-panel>div.widget:last-child") get the last <div> with class widget within scroll-panel

Basics, do something with it : Methods

$(...).css("background","pink")

$(...).fadeIn()

$(...).append("<p>Bonjour le monde</p>")

$(...).click(function(){alert("click");})

  • Bonus : you can chain the calls : $(...).slideDown().val("slider")
  • Also : one method, many uses :
    • $("#myinput").val()
    • $("#myinput").val("zero")
    • $("#myinput").val(function(index,value){})
